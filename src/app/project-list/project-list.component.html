ToPlayList API REST inspirada pelo site backloggd.com

API REST criada com Spring Web, Spring Security,
Spring Data JPA e PostgreSQL com a finalidade de
organizar os jogos que você planeja jogar. Foi
inspirada no site backloggd.com, que tem um
proposito similar, além de possuir outras
features que não foram implementadas aqui.
Loto-Gen API para gerar números de loteria aleatórios

API REST criada com Spring Web, Spring Security,
Spring Data JPA e PostgreSQL com a finalidade de
gerar números aleatórios para alguns tipos de
loterias brasileiras. Possui autenticação OAuth2
e armazenamento de um histórico de números
gerados

Capivara Finance API REST para controle de finanças pessoais

API criada com Spring Boot, Spring Security e
PostgreSQL, além de JUnit e Mockito para os
testes unitários. Fornece funcionalidades como
cadastro de despesas, ganhos e filtragem de
dados cadastrados. Também possui autenticação
com Token JWT.
